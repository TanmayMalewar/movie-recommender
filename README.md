# 🎬 Movie Recommendation System

This is a Machine Learning project that recommends movies based on similarity of content.

## 🔧 Technologies Used
- Python
- Pandas
- Scikit-learn

## 📌 How it works
It uses cosine similarity on movie features (tags) to find similar movies.

## 🚀 How to run
1. Open the Jupyter Notebook (.ipynb file)
2. Run all cells step by step
3. Enter a movie name to get recommendations

## 💡 Example
Input: Batman Begins  
Output: The Dark Knight, Joker, Inception, etc.
